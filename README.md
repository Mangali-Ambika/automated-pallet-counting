# Automated Pallet Counting

YOLOv8-based computer vision solution to automatically detect and count pallets in warehouse environments. Reduces manual labor by **70%** and minimizes human errors in inventory tracking.

## ✨ Features
- Real-time pallet detection and counting using YOLOv8
- Automated inventory tracking system
- High-accuracy model trained on custom dataset
- Easy-to-use Python script for inference
- Dataset collection and annotation workflow

## 🛠 Tech Stack
- **Model**: YOLOv8 (Ultralytics)
- **Languages**: Python
- **Libraries**: OpenCV, Roboflow, NumPy, Matplotlib
- **Dataset Tool**: Roboflow (annotation + versioning)
- **Environment**: Google Colab / Jupyter Notebook

## 🚀 How to Run Locally
```bash
git clone https://github.com/Mangali-Ambika/automated-pallet-counting.git
